# Theoretical Framework

The exocortex is based on the idea that thinking does not occur only
inside the brain but within a larger cognitive environment.

This environment includes:

- external memory
- symbolic structures
- visual representations
- computational tools

Together they form a cognitive ecosystem.

---

## Cognitive field

Ideas are not treated as isolated elements but as nodes inside
a dynamic field.

In this field:

- relations appear
- densities form
- attractors emerge

This makes it possible to observe structures that would remain
invisible in linear notes.

---

## Emergence

The system is designed to allow structures to emerge from material
rather than forcing predefined categories.

Patterns become visible through:

- repetition
- proximity
- conceptual resonance

---

## Human decision

The system does not autonomously consolidate structures.

The final step always remains human:

recognition → validation → stabilization

This prevents uncontrolled recursion and maintains the role
of human judgment.
