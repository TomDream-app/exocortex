# Context

The exocortex uses context to orient cognitive activity.

Context defines the current environment of thinking.

---

## Static context

Long-term elements.

Examples:

- research themes
- conceptual frameworks
- personal knowledge domains

These elements evolve slowly.

---

## Dynamic context

Short-term orientation.

Examples:

- current project
- active question
- exploration session

Dynamic context changes frequently.

---

## Minimal context structure

A minimal context description can include:

AXE  
PROJECT  
MEMORIES  
SESSION OBJECTIVE

This structure allows the system to remain flexible
while still providing orientation.
