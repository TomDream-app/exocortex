# Architecture

The exocortex architecture is based on five fundamental modules.

capture → memory → transformation → structure → action

---

## Capture

Collection of raw material.

Examples:

- notes
- links
- images
- transcripts
- conversations

All captured elements enter the system as raw data.

---

## Memory

Raw material is stored in a structured archive.

Memory remains flexible and does not enforce strong categorization.

---

## Transformation

Material is processed to reveal potential relations.

This stage may include:

- embeddings
- clustering
- semantic analysis
- tagging

---

## Structure

Patterns discovered during transformation can be consolidated
into explicit structures.

These structures become stable reference points.

---

## Action

The final goal is to support decision and creative production.

Structures guide future exploration and new capture processes.
