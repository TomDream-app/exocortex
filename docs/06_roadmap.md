# Roadmap

The development of the exocortex follows several phases.

---

## Phase 0

Publication of the conceptual framework.

Repository creation and architecture definition.

---

## Phase 1

Stabilization of the core system.

Definition of:

- capture
- memory
- transformation
- structure
- action

---

## Phase 2

Implementation of operational components.

Especially:

log extractor  
context engine

---

## Phase 3

Development of first application.

Visual Exocortex.

A system designed to transform large generative image archives
into navigable cognitive landscapes.

---

## Phase 4

Expansion of the ecosystem.

Additional applications and tools built on the exocortex core.
