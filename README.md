# EXOCORTEX

Exocortex is an external thinking space.

It is designed as a cognitive environment where ideas, memories and materials
are captured, structured and transformed into evolving architectures of thought.

The goal is not only to store information but to allow structures to emerge
from accumulated material and guide decision.

---

## Core principle

capture → memory → transformation → structure → action

The system treats knowledge not as static information but as a dynamic field
where patterns, attractors and structures can emerge.

---
